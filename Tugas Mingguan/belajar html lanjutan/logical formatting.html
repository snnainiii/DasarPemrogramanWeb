Daftar tag untuk pengaturan teks secara logis

CITE : menyatakan teks adalah kutipan atau citation

CODE : menyatakan bahwa teks adalah kode tau program komputer

EM : menyatakan penekana pada teks. umumnya teks akan ditampilkan miring

KBD :menyatakan teks adalah masukan dari keyboard

SAMP : menyatakan bahwa teks adalah kontanta sederetan karakter

STRONG : menyatakan bahwa teks yang sangat penting untuk diperlihatkan orang

VAR : menyatakan teks adalah nama variabel atau ekspresi matematika atau program komputer
